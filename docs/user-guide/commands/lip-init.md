# lip init

This command can be used to set up a new or existing lip package

## Usage

```
lip init [OPTIONS]
```

## Options

- `-h, --help`: Prints help information
- `-q, --quiet`: Show only errors
- `-v, --verbose`: Show verbose output
- `-f, --force`: Overwrite the existing tooth.json file
- `--init-avatar-url <URL>`: The avatar URL to use
- `--init-description <DESCRIPTION>`: The description to use
- `--init-name <NAME>`: The name to use
- `--init-tooth <TOOTH>`: The package's tooth path to use
- `--init-version <VERSION>`: The version to use
- `-y, --yes`: Skip confirmation prompts
