# lip init

Create a tooth.json file

## Usage

```shell
lip init
```

## Description

This command can be used to set up a new lip package.

lip will silently create a default `tooth.json` for you. If a `tooth.json` already exists, the command will fail.

## Examples

Basic initialization:

```shell
# Create a new project with default values
lip init
```

## Options

None.
